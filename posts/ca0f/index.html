mac常用快捷键

mac

编辑

复制、剪切、粘贴

复制：command + c
剪切：command + x
粘贴：command + v

撤销、重做

撤销：command + z
重做：command + shift + z

全选、取消选择

全选：command + a
取消选择：command + shift + a

查找替换

查找：command + f
替换：command + option + f

窗口快捷键

1、切换应用程序

command + tab

2、最大化窗口

最大化当前窗口：command + control + F
最大化所有窗口：option + 点击窗口最大化按钮

3、窗口移动、重置大小

窗口移动：command + 鼠标拖拽窗口
重置大小：option + 点击红色关闭按钮

系统快捷键

截屏、录屏

截全屏：command + shift + 3
自定义截屏：command + shift + 4
录屏：command + shift + 5

command + n

command + t

command + shift + [

command + shift + option + ]
